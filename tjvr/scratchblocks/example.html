scratchblocks2 example

Example static usage of scratchblocks2.

    define dance (speed)
    set [dist v] to ((speed) * (distance to [mouse-pointer v]))
    repeat until ((timer) > [10])
        move (dist) steps
        wait (1) secs // Pause for a moment
        move (() - (dist)) steps
        play note (42 v) for (0.5) beats
        if <mouse down?>
            say [Whoops!]
        end
    end
